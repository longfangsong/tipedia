# 迁移组件的数据目录

1. 使用 `tiup cluster stop <集群名>` 停止集群
2. 修改集群配置文件 `meta.yaml`，建议将 `meta.yaml` 中 `global` 和各组件的 `data_dir` 都调整掉
3. 复制老目录的所有数据文件到新目录下
4. 删除 tikv 的数据目录中的 last_tikv.toml
5. reload 集群
  ```shell
  tiup cluster reload <集群名> --force
  tiup cluster stop <集群名>
  tiup cluster start <集群名>
  ```
